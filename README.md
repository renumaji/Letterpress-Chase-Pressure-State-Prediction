# Letterpress Chase Pressure State Prediction

From-scratch Eris challenge. Solvers reconstruct final letterpress chase zone state by executing a custom lockup trace VM and learning measurement calibration.

Upload `letterpress-chase-pressure-vm-raw.zip` as the raw dataset. Use `problem.md`, `grade.py`, `prepare.py`, `config.yaml`, and `rubrics.yaml` in the challenge editor.
